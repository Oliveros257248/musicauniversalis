Key Terms · Probability & Statistics Glossary
Poisson Distribution
Models the number of events occurring in a fixed interval when events happen independently at a constant average rate lambda. Formula: P(n) = lambda^n * e^(-lambda) / n!. Named after Simeon Poisson (1837). As t grows small relative to the time constant, the binomial distribution converges to Poisson.
Binomial Distribution
Models the number of successes in M independent yes/no trials each with probability p. Formula: P(n) = C(M,n) * p^n * (1-p)^(M-n). Here M = 100 molecules, each converting independently with probability p = 1 - e^(-t/Mτ).
Gamma Function: Γ(n)
Generalizes factorial to non-integer values: Γ(n+1) = n! for integers, but also defined for real n. Γ(0.5) = sqrt(pi). Allows the distributions to be evaluated at fractional state values (like n = 1.013643 or n = 3.013643), which is what the Phi perturbations produce on this page.
Law of Rare Events
When M is large and p is small but Mp is finite, the binomial distribution converges to Poisson with lambda = Mp. Mathematically: C(M,n) * p^n ≈ (Mp)^n / n! as p → 0, M → infinity. Demonstrated in Chart 1: ratio R = P_Poisson / P_Binomial → 1 at t ≈ 20.
Sequential Poisson Process
A chain of Poisson steps: A → B → C → ... where each conversion takes an average waiting time τ. The probability of being in state D (three steps forward) at time t is a Poisson distribution with parameter λ = t/τ evaluated at n = 3. Generalizing to real n via Γ(n+1) replaces the factorial.
Φ (Phi) Perturbation
On this page, Phi = (3/2)^12 / 2^7 = 1.013643 (the Pythagorean comma ratio). Used as a perturbation: alpha = Phi - 1 = +0.013643, beta = -alpha = -0.013643. Applied to state n=3 to generate n = 3.013643 (P1) and n = 2.986357 (P2). See CF flag below for the interpretive note.
Lanczos Approximation
A numerical method for computing Γ(n) to arbitrary precision for real n. Used in the JavaScript implementation on this page to evaluate the generalized Poisson/Binomial formulas at non-integer states. Accurate to 15 significant figures for most inputs.
Perturbation Analysis
Studying how a small change to a parameter (here, a change of alpha = 0.013643 in the state index n) affects the output. If the ratio P(n+alpha)/P(n) is close to 1, the system is insensitive to this perturbation. Chart 4 shows this sensitivity as a function of time.

1 · Background & Problem Statement

PART A
Sequential Poisson Process
A → B → C → … → N with average waiting time τ between steps. Starting at t=0, what is the probability of observing the system in state D (three forward steps) at time t?
POISSON: GENERALISED TO REAL n VIA GAMMA FUNCTION
PPoisson(n, t) = [(t/τ)ⁿ / Γ(n+1)] · e−t/τ
For integer n=3 (state D): Γ(4) = 3! = 6
PART B
M Independent Molecules
M molecules A1…AM each independently convert to B with time constant Mτ. Starting with all in state A at t=0, probability of exactly n in state B at time t?
BINOMIAL: GENERALISED TO REAL n VIA GAMMA FUNCTION
PBinomial(n, t) = [Γ(M+1) / (Γ(n+1)·Γ(M−n+1))] · pn · (1−p)M−n
where p = 1 − e−t/(Mτ) (single-molecule conversion probability)
PART C
Convergence in the t ≪ Mτ Limit
Show that as t ≪ Mτ, PBinomial → PPoisson. The law of rare events: each molecule converts rarely (p ≈ t/Mτ ≪ 1), but collectively they produce a finite Poisson rate λ = Mp = t/τ.
KEY APPROXIMATIONS (t ≪ Mτ)
p ≈ t/(Mτ)  ·  C(M,n) ≈ Mⁿ/n!  ·  (1−p)M ≈ e−t/τ
→ PBinomial ≈ (t/τ)ⁿ/n! · e−t/τ = PPoisson ✓

2 · The Three Φ Definitions

⚐ COMMA FRAMEWORK ARTICULATION · The choice of Phi = 0.013643 as a perturbation is mathematically valid but the reason for choosing this specific value is interpretive, not derived from probability theory.
Why these three values of Phi?
The standard mathematical approach would choose an arbitrary small epsilon for perturbation analysis. This study uses three specific values that come from the Pythagorean comma framework: Phi = (3/2)^12 / 2^7 = 1.013643 (the exact comma ratio), and two related values constructed from (3/5) ratios. The perturbation alpha = 0.013643 is genuinely tiny, comparable to rounding errors in floating-point arithmetic, which is why P1 and P2 track so closely to the n=3 baseline. The mathematical analysis (convergence, law of rare events, Gamma function generalization) is entirely standard and correct regardless of which perturbation value is chosen. The choice of this value is a comma framework observation, not a mathematical necessity.

Three values of Φ are used to define perturbation parameters α = Φ−1 and β = −α, generating four n-scenarios: n=3 (baseline), n=Φ, n=3+α (P₁), n=3+β (P₂).

5 · Analysis of Part (c) & Results

Part (c) states that when t ≪ Mτ, the Binomial answer from part (b) reduces exactly to the Poisson answer from part (a). The mechanism is the law of rare events; each of the M molecules has a tiny individual conversion probability p ≈ t/Mτ, but collectively they produce a finite expected count Mp ≈ t/τ, which is precisely the Poisson rate. Chart 1 confirms this precisely: the ratio R = PPoisson/PBinomial hits 1.000 at t ≈ 20 and stays within 5% for t < 50. The fitted trend line R(u) = 1.137·e−1.361u − 0.107 quantifies how fast the equivalence breaks down as u = t/Mτ grows, degrading faster than linearly once u exceeds ~0.05.

The three Φ definitions reinforce this. For Φ ≈ 1.014 (α tiny), all scenarios collapse onto each other; the equivalence is robust to small perturbations in n. For ø and ø₂ (α ≈ −1), P₁ and P₂ probe n ≈ 2 and n ≈ 4, yet the Poisson/Binomial ratio stays similarly ~0.96 at t = 50 for both, meaning the convergence quality is largely independent of which state you are near. The approximation breaks down for the same physical reason in all cases: at large t the A pool depletes, suppressing further conversions in the Binomial but not in the memoryless Poisson.

GAMMA FUNCTION: LANCZOS APPROXIMATION
function gamma(z) {
  if (z < 0.5) return Math.PI / (Math.sin(Math.PI * z) * gamma(1 - z));
  z -= 1;
  const c = [0.99999999999980993, 676.5203681218851, ...];
  let x = c[0];
  for (let i = 1; i < 9; i++) x += c[i] / (z + i);
  const t = z + 7.5;
  return Math.sqrt(2π) · tz+0.5 · e−t · x;
}
